# Equicksales-Apache-Access-LogAnalyzer
This project is about analyzing apache access log. We have been conducting research on developing a tool for apache access log analysis. This research project has culminated into the development of a software for analyzing apache access log files.</br>
To test this tool download the project files and extract it. Then copy the dist folder in the LogAnalyzerGUIApp. You can double click the LogAnalyzerGUIApp.jar file and the tool will launch.
To be able to download the apache access log file from a remote LAMP server in order to analyze it using the log analyzer GUI app, you need to install Java on the LAMP server and copy the FileTransferServerLib.java file in the Server Script folder onto your LAMP server. Then run the file. You can also make sure that the FileTransferServerLib.java file is always running on the remote LAMP sever. In that case you will need to update the FileTransferServerLib.java file, with an unending while loop in the main method of the file then Create a systemd Service for the FileTransferServerLib.java app. That way the app on the LAMP server will always be listening to the GUI App for connections to allow you download apache access log file.
